## GL-MT2500/GL-MT2500A

![gl-mt2500 interface](https://static.gl-inet.com/docs/en/4/user_guide/gl-mt2500/hardware_info/mt2500_interface.jpg){target="_blank"}

* System
	* Blue: system booting
	* White: system working normally
* VPN
	* Off: VPN inactive
	* On: VPN active
